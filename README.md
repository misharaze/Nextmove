 NextMove — Лендинг для компании по переездам

Современный одностраничный сайт для компании, предоставляющей услуги квартирных, офисных и междугородних переездов.  
Проект выполнен в неоново-футуристическом стиле с полной адаптацией под мобильные устройства, а также с рабочей формой обратной связи через Node.js и Vercel.



 Возможности проекта

- Современный дизайн в стиле digital  
- Полностью адаптивная верстка (ПК, планшеты, телефоны)  
- Бургер-меню для мобильных устройств  
- Модальное окно заявки  
- Отдельная секция контактной формы  
- Анимации появления блоков  
- Карусель отзывов  
- FAQ с анимацией (вопрос–ответ)  
- Интеграция с почтой через Nodemailer  
- Backend через Serverless Functions (Vercel)  
- SEO-настройка + OpenGraph  
- Бесплатный HTTPS  
- Готов к продакшену  



 Используемые технологии

- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- Bootstrap 5  
- Node.js (serverless)  
- Nodemailer  
- Vercel  
- GitHub
